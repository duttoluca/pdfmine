INSTALLAZIONE
Installare Python 2.7.x
Installare pip (http://pip.readthedocs.org/en/latest/installing.html)
Installare le librerie necessarie: pip install -r requirements.txt

ESECUZIONE
Lo script necessita come parametro il path di una cartella.
In tale cartella leggerà i PDF e scriverà un file index.txt, con formato
NOMEFILE;protocollo
Esempio
python pdfscrape.py c:\test\pdf
